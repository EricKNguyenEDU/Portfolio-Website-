Creating a Game with Code

For my game, I wanted to create a very interactive experience. I came up with the idea of having the player paint their own canvas. I designed a game where the player clicks on white circles to “pop” them, causing each circle to change into a random color. The more the player clicks a circle, the more the color changes. Additionally, the background changes to a random color, making the overall experience more engaging.
